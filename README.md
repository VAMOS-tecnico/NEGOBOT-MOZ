# NEGOBOT-MOZ
O Negobot Moz é uma plataforma SaaS de chatbot inteligente para atendimento ao cliente, desenvolvida especificamente para o mercado moçambicano. A solução permite às empresas automatizar o atendimento 24 horas por dia, 7 dias por semana, com integração nativa no WhatsApp, Facebook Messenger e websites.
